    AI 助手装机准备清单

    准备好以下材料，装机当天 40 分钟搞定。

    你要准备的东西

    第一步：买一台云服务器（10 分钟）

    AI 助手需要一台 24 小时运行的服务器。有两个选择：

    选项 A：国内服务器（便宜，推荐新手）

    推荐火山引擎（字节跳动家的，跟飞书同一家）：

    1. 打开 火山引擎官网，用手机号注册
    2. 搜「轻量级云服务器」或「ECS 云服务器」
    3. 选最便宜的：2 核 2G，约 29 元/年
    4. 系统选 Ubuntu 24.04
    5. 付款（支持支付宝/微信）
    6. 买完后记下：服务器 IP 地址 和 登录密码

    其他选择

    腾讯云、阿里云也行，配置要求：1 核 2G 内存以上，Ubuntu 系统。

    选项 B：海外服务器（搜索更强，贵一点）

    推荐 Vultr：

    1. 注册 Vultr 账号（可以找帮你装机的人要推荐码，双方各得优惠）
    2. 选新加坡或东京节点，约 $6/月（~43 元）
    3. 系统选 Ubuntu 24.04
    4. 支持信用卡 / PayPal

    海外服务器的好处

    搜索能力更强、访问海外网站无障碍。适合对搜索质量有要求的用户。

    第二步：准备飞书（5 分钟）

    • [ ] 手机装好飞书 App，并且已经注册登录
    • [ ] 记住你飞书绑定的手机号（配置时要用）

    第三步：选 AI 模型

    AI 助手的「大脑」有几种选择：

    模型 价格 特点 你需要做什么
    Deepseek（国内） ~5 元/月 便宜、中文好 注册 deepseek.com 充 10 块
    ChatGPT（海外） ~15 元/月 综合最强 帮你装机的人搞定，不用操心

    不确定选哪个？

    跟帮你装机的人说你主要用来干嘛，让 ta 帮你选。不确定就先选 Deepseek，随时可以换。

    第四步：想好这些问题

    • [ ] 机器人叫什么名字？（比如：小助手、小六、阿豆……随便起）
    • [ ] 你主要用它干什么？（帮你选配置）：
      • A. 只聊天 + 搜索（最简单）
      • B. 聊天 + 搜索 + 日历 + 待办 + 写文档（推荐，大多数人选这个）
      • C. 上面全部 + 帮你给家人发消息 + 查联系人（完整版）
    • [ ] 你在哪个城市？（查天气用的，默认深圳）

    如果你选了 C（完整版）

    • [ ] 需要它联系的家人的手机号（飞书绑定的那个）
    • [ ] 家人也要装飞书（可以之后再装，不急）

    配置当天需要你做的

    • [ ] 在旁边，有几个地方需要你手机扫码/点确认
    • [ ] 你的飞书账号不要开两步验证（如果开了，配置时需要你帮忙输验证码）
    • [ ] 预留 40-60 分钟（首次含服务器初始化）

    费用一览

    最省方案（纯国内）

    项目 费用 说明
    云服务器 ~2.5 元/月 火山引擎 29 元/年
    AI 模型 ~5 元/月 Deepseek，自己充值
    飞书 免费 个人版即可
    搜索 API 免费 装机时配好
    合计 ~8 元/月

    推荐方案（国内服务器 + 海外模型）

    项目 费用 说明
    云服务器 ~2.5 元/月 火山引擎 29 元/年
    AI 模型 ~15 元/月 ChatGPT
    飞书 免费 个人版即可
    搜索 API 免费 装机时配好
    合计 ~18 元/月

    最强方案（纯海外）

    项目 费用 说明
    云服务器 ~43 元/月 Vultr
    AI 模型 ~15 元/月 ChatGPT
    飞书 免费 个人版即可
    搜索 API 免费 装机时配好
    合计 ~58 元/月 搜索最强，体验最好

    配好之后你能干什么

    所有人都有的能力（A/B/C 通用）

    你说 它做什么 举例
    聊天 回答任何问题，像微信聊天一样 「帮我想个生日礼物」
    搜索 同时搜十几个搜索引擎，总结给你看 「最近有什么好看的电影」
    读网页 发一个链接，它读完帮你总结 「帮我看看这篇文章讲了啥」
    查天气 今天穿什么、要不要带伞 「明天深圳天气怎么样」
    记东西 说过的事情它会记住，下次还知道 「记住我对虾过敏」
    群聊 拉到群里，@它 就回复，不 @ 不说话 在家庭群 @小六 问问题

    免费能力

    搜索、天气、网页阅读这些能力装机时一并配好，全部免费，不需要额外花钱。

    选了 B 多出来的能力

    你说 它做什么 举例
    建日程 在飞书日历上建事件，到点提醒你 「明天下午3点开会」
    建待办 帮你记任务，飞书里能看到清单 「记一下周五交报告」
    写文档 直接在飞书里创建文档 「帮我列个出差清单」
    搜文档 搜你飞书里之前的文档 「帮我找之前那个会议纪要」

    选了 C 再多出来的能力

    你说 它做什么 举例
    发消息 替你给家人发飞书消息 「跟老公说晚上别做饭了」
    查联系人 查组织里的人 「查一下小王的手机号」

    关于「授权」

    选了 B 或 C 的话，第一次用每个新功能时，飞书会弹一个授权按钮：

    • 第一次建日程 → 弹出来 → 你点「授权」
    • 第一次建文档 → 弹出来 → 你点「授权」
    • ……每个功能只弹一次

    点一次就永久生效，以后用就不弹了。这是飞书的正常安全机制，问你要什么权限就点同意。

    选 A 的话没有这个过程，开箱即用。

    常见问题

    Q：最便宜要多少钱？
    纯国内方案约 8 元/月。推荐方案约 18 元/月。
    Q：服务器是干嘛的？
    AI 助手需要一个 24 小时开机的电脑来运行，云服务器就是你在网上租的一台小电脑。
    Q：国内和海外服务器有什么区别？
    海外服务器搜索能力更强（能搜到更多内容），但贵一些。国内服务器便宜稳定，日常够用。
    Q：Deepseek 和 ChatGPT 有什么区别？
    Deepseek 便宜、中文好；ChatGPT 综合更强、英文好。不确定就选 Deepseek 先用着，随时可以换。
    Q：它会看到我的聊天记录吗？
    不会。它只能看到你主动发给它的消息。数据存在你自己的服务器上。
    Q：我可以删掉它吗？
    随时可以，在飞书里删除对话就行。不想要了把服务器退了就彻底没了。
    Q：它会主动打扰我吗？
    不会，只有你找它说话它才回。
    Q：群里它会每条消息都回吗？
    不会，只有 @它 的时候才回。
    Q：火山引擎注册需要实名吗？
    需要，国内云服务都要实名认证。支付宝认证就行，几分钟的事。
    Q：Vultr 要信用卡吗？
    是的，支持 Visa/Mastercard 或 PayPal。
